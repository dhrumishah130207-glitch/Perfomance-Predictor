# PredictEdu AI — Student Performance Intelligence Engine

A Flask-based web app that predicts student academic performance using a trained ML model (RandomForest / GradientBoosting), with a cyberpunk-themed UI, full admin panel, and CSV-based data persistence.

---

## 📁 Project Structure

```
predictedu/
├── app.py                      # Flask backend (routes, ML, CSV logic)
├── train_model.py              # ML training script
├── model.pkl                   # Trained model (generated by train_model.py)
├── Student_Performance_Dataset.csv   # Training dataset
├── student_records.csv         # User session records (cleared by user)
├── admin_all_records.csv       # Admin permanent log (never cleared)
├── requirements.txt
├── README.md
└── templates/
    ├── index.html              # Main app (Home / Predictor / Dashboard / Records)
    ├── admin_login.html        # Admin login page
    └── admin.html              # Admin dashboard
```

## ⚙️ Setup

### 1. Install dependencies
```bash
pip install -r requirements.txt
```

### 2. Train the ML model
```bash
python train_model.py
```
This reads `Student_Performance_Dataset.csv`, trains and compares RandomForest vs GradientBoosting,
picks the best, and saves `model.pkl`.

**Required CSV columns:**
- `Study_Hours_Per_Day`
- `Attendance_Percentage`
- `Previous_Year_Score`
- `Math_Score`
- `Science_Score`
- `English_Score`
- `Final_Percentage` (target)

If columns are missing or the CSV isn't found, the script auto-generates synthetic training data.

### 3. Run the app
```bash
python app.py
```
Open: http://localhost:5000

## 🤖 ML Model

**Features used:**
| Feature | Description |
|---|---|
| Study_Hours_Per_Day | Daily study time |
| Attendance_Percentage | Class attendance % |
| Previous_Year_Score | Prior academic score |
| Math_Score | Math subject score |
| Science_Score | Science subject score |
| English_Score | English subject score |

**Target:** `Final_Percentage`

**Models compared:** RandomForestRegressor vs GradientBoostingRegressor → best R² wins.

If `model.pkl` is missing, app falls back to a weighted formula.

## 📊 Dataset

`Student_Performance_Dataset.csv` — 5,000 student records with:
`Student_ID, Age, Gender, Class, Study_Hours_Per_Day, Attendance_Percentage, Parental_Education, Internet_Access, Extracurricular_Activities, Math_Score, Science_Score, English_Score, Previous_Year_Score, Final_Percentage, Performance_Level, Pass_Fail`
